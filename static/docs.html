Legal Cortex Technical Documentation

Comprehensive guide for advocates, legal researchers, and system administrators operating Legal Cortex Enterprise.

Confidentiality Guarantee: Legal Cortex runs 100% locally via Ollama. All document chunking, embeddings, and chat inference occur on your machine or private firm server. No external network requests are executed.

System Purpose

Legal Cortex transforms standard hardware into a high-speed legal workstation for Indian Jurisprudence. It automates procedural pleading drafts, interprets complex statutes, evaluates case facts for litigation win probabilities, and synthesizes Supreme Court precedents.

Quickstart Guide

  1. Launch Local Ollama Service

    Ensure Ollama is active on http://localhost:11434 and pull required models:

    ollama pull qwen2.5:3b
ollama pull nomic-embed-text
  2. Start Backend Application

    Activate python environment and run server script:

    python3 backend.py
  3. Access Legal Workspace

    Navigate to http://127.0.0.1:8000/index.html in your browser.

Constitution of India Auto-Ingestion

Upon server launch, an asynchronous background thread automatically parses Data/Constituion Of India/Constitution Of India.pdf. This loads fundamental rights, directive principles, and writ jurisdiction into memory without blocking web server startup.

Drafting Engine Guide

The Drafting Engine utilizes specialized prompts forcing Indian courtroom formatting (e.g. "IN THE COURT OF...", prayer clauses, and undertaking affidavits).
